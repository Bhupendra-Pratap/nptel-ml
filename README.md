# NPTEL ML Prep

A simple frontend app to review and practice NPTEL Machine Learning assignment questions.

## Files

- `index.html` - Main page structure
- `styles.css` - App styling
- `script.js` - Question data and app logic

## How to run

1. Open `index.html` in your browser.
2. Select a week from the top navigation.
3. Use **Read** mode to view answers or **Practice** mode to test yourself.

## Notes

- No build step is required.
- This is a static HTML/CSS/JavaScript project.
